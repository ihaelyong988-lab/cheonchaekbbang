# v1.8 방문자 플로우 개선 백로그 (2026-07-19 실사 확정)

방문자 관점 7차원 실사(Workflow: finder 7기 실측 + 병합 + 적대 검증)로 발굴한 확정 개선안 27건.
상세 전문(증상·근거·처방·채점 절차)은 상위 폴더 `AGENTS.md` §10 및 `docs/visitor-audit-20260719/` 원본 JSON 참조.
구현 시 이 순번대로, worktree 격리 병렬 구현 → 항목별 1:1 매칭 검증 → `npm test` + CI quality 통과 후 머지한다.

## 기능 유무 확인 결과 (실측 통과)
- 홈탭: 첫 화면 랜딩, 질문 카드, 수집, 다른 질문, 질문 검색, 수치 스트립 4링크, 분야 게이지-서재 연동 전부 작동.
- 닫기 팝업창: 있음 + 전 시나리오 작동(뒤로가기 팝업, 계속 읽기, ESC, 반복 뒤로가기 방어, 포커스 트랩, 닫기 폴백, 데이터 보존). 탭 X 닫기 보호는 불필요 판정(저장이 전부 즉시형 + pagehide 플러시로 유실 0).
- 내 서재 저장, 기록 유지, 재방문 복원 정상.
